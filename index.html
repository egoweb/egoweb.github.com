EgoWeb

Egoweb is a tool for social network interviewing. It built off of Chris McCarty’s Egonet. EgoWeb 1.0 is a java app for collecting and processing personal networks.

It improved on Egonet's interviewing capabilities. EgoWeb 2.0 is a conversion of the java app into a PHP application.

EgoWeb 2.0 can do a variety of type of network data collection beyond personal networks, including whole networks and cognitive networks.
