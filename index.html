EgoWeb

The page has moved to: https://github.com/qualintitative/egoweb

Egoweb is a tool for social network interviewing. It initially built off of Chris McCarty’s Egonet which is a tool for collection of personal network data. EgoWeb now facilitates colletion of a variety of types of social network data.
